🎯 Option 1 - Utiliser directement (Recommandé) :

// Intégrer via iframe <iframe src="https://www.ccom.ucsd.edu/~cdeotte/programs/MNIST.html" width="800" height="600"></iframe> // Ou rediriger vers la démo window.open('https://cs.stanford.edu/people/karpathy/convnetjs/demo/mnist.html');

🔧 Option 2 - Récupérer le code source :

// ConvNetJS (Open Source) git clone https://github.com/karpathy/convnetjs cd convnetjs/demo/ // Ou télécharger directement les fichiers JS wget https://cs.stanford.edu/people/karpathy/convnetjs/build/convnet.js // Intégrer dans ton projet <script src="convnet.js"></script>

🚀 Option 3 - Créer ta propre API :

// Utiliser TensorFlow.js avec un modèle pré-entraîné const model = await tf.loadLayersModel('URL_DU_MODELE/model.json'); // Fonction de prédiction async function predictDigit(canvas) { const tensor = tf.browser.fromPixels(canvas, 1) .resizeBilinear([28, 28]) .div(255.0) .expandDims(0); const prediction = model.predict(tensor); const result = prediction.argMax(1).dataSync()[0]; return result; }
