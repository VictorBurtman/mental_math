🎮 Comment jouer

Règles du Sudoku :

  • Remplir une grille 9×9 avec les chiffres 1 à 9
  • Chaque ligne doit contenir tous les chiffres 1-9
  • Chaque colonne doit contenir tous les chiffres 1-9
  • Chaque région 3×3 doit contenir tous les chiffres 1-9

Utilisation de MNIST :

  • Dessinez votre chiffre dans la zone MNIST
  • L'IA va le reconnaître avec 98.5% de précision
  • Utilisez le résultat pour remplir les cellules
  • Parfait pour tester la reconnaissance de chiffres !
